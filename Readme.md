Dockerで開発に必要なRedmine、Jenkins、GitBucker、MySQLサーバ環境を一気に構築します。  

動作環境は以下の通りです。

```
docker: Docker version 1.13.1, build 092cba3
docker-compose: docker-compose version 1.11.1, build 7afaa436
```


docker-compose.ymlを適当に編集後  

docker-compose up -d  

してください。
